Natural Language Interfaces
===========================

C-Phrase
https://code.google.com/p/c-phrase/
- Natural Language Interface to Postgre / ODBC databases.
- English language support
- Written in Lisp, with a PHP frontend
- 2009 - 2011

TEAM
TEAM: An experiment in the design of transportable natural language interfaces (1987)
- benadert willekeurige databases

ORAKEL
Towards portable natural language interfaces to knowledge bases – The case of the ORAKEL system (2008)
- benadert willekeurige databases

Online Q & A
============

Evi
http://www.evi.com/
- via an API (simple question-answer only)
- api: http://developer.evi.com/download/
- forum: http://support.evi.com/forums

Wolfram Alpha
www.wolframalpha.com/
- api: http://products.wolframalpha.com/api/documentation.html
- forum: http://community.wolfram.com/content?curTag=software%20development

START
http://start.csail.mit.edu/
START, the world's first Web-based question answering system, has been on-line and continuously operating since December, 1993.
- No access to source code, but many articles describing it

Watson (IBM)
- via an API (simple question-answer only)
- api: https://www.ibmdw.net/watson/docs/category/access-the-question-and-answer-api/
- forum: https://www.ibmdw.net/answers?community=watson

Quepy
https://github.com/machinalis/quepy
Quepy is a python framework to transform natural language questions to queries in a database query language.
- online demo: http://quepy.machinalis.com/
- Works with a fixed set of expressions
